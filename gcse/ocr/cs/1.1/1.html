1.1.1 - Architecture of the CPU

Factors that effect CPU performance

- cache size
- cores
- architecture
- clock speed


3 parts of a computer

- Keyboard (Input)
- Monitor (Output)
- CPU (Processes)


4 parts to a CPU

- Cache
- ALU (arithmetic and logic unit)
- CU (control units)
- Registers


Registers

- Program counter: tracks current execution point
- Instruction register: stores instructions soon to be executed
- Memory address register: stores memory addresses
- Memory buffer register: buffers data going to and from memory
- Accumulator register: ALU results storage
